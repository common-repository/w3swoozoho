Using WooCommerce to Zoho CRM plugin create Contact and Account in your Zoho CRM automatically when order placed in WooCommerce.

== Description ==

This plugin provides you that automated sync with your Zoho CRM and WooCommerce. Whenever there is a sale, it will insert the customer's data into the CRM as a Contact/Account with the product information.

= Features List: =
1. Add customers as contacts in Zoho CRM when they place an order in your WooCommerce store.
1. Update contacts/customers with existing records in Zoho CRM.
1. Real time sync.
1. Duplicate customer check by email.
1. Product details in Notes if free version.
1. Sync Logs (coming soon)
1. Support Zoho Rest API v2
1. Regular enhancements via updates.
1. Support all Zoho Data center [.COM/.EU/.IN etc].
1. Quick and easy setup.

== Installation ==

Please follow the instruction below:

1. Upload the plugin files to the `/wp-content/plugins/w3swoozoho` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the "W3SWooZoho"->settings screen to configure the plugin
1. Create Zoho Client Credentials.
1. Populate settings form and Save.
1. Click the Authorize button.
1. On Zoho authentication screen, click the green Grant button.
